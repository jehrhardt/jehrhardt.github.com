Emacs as primary text editor for lazy developers

Jan 8th, 2013

I use Emacs as my default text editor for years now. Similar to oh-my-zsh there is a simple setup for Emacs.

On Mac OS X this will install the latest version of Emacs and Emacs Prelude. Prelude is a ready to use configuration for Emacs, that gives you all you need as a programmer.

Emacs will load its whole configuration, when you start it. This makes startup very slow, especially when you use a lot of modes like in Prelude. Speed up Emacs start time can be done by running it as daemon.
